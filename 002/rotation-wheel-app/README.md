# Rotation Wheel App

## Overview
The Rotation Wheel App is a simple web application that features a rotating wheel to randomly select one of two names. It provides an interactive and fun way to make decisions or choose between options.

## Features
- A visually appealing rotating wheel that spins to select a name.
- User-friendly interface with buttons to start the rotation.
- Smooth animations for an engaging user experience.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)

### Running the Application
1. Clone the repository or download the project files.
2. Open the `src/index.html` file in your web browser.
3. Click the "Spin" button to start the rotation wheel.

### Customization
- You can modify the names in the JavaScript file (`src/js/script.js`) to change the options available for selection.
- Adjust the styles in `src/css/styles.css` to customize the appearance of the wheel and buttons.
